# 密集对话采样补充

## 问题

`sample_subtitles.py` 对密集对话视频的采样输出过于稀疏，不足以把握核心论点。原因：

- 对话视频字幕段极短（~1-1.5s/段），40 分钟视频产生 1500-2000 条段
- 采样算法等距选取后每条单独展示，丢失上下文连续性
- 最终输出 ~50 条孤立文本片段，难以还原论点和论证链

## 解决方案：时间区间文本聚合

采样建立结构骨架后，用 Python 按时间窗口聚合完整文本块做补充：

```python
import json

with open('/tmp/subs.json') as f:
    subs = json.load(f)

# 按主题边界定义时间窗口（从采样大纲获取）
ranges = [
    (0, 180, "开场 & 背景"),
    (180, 420, "核心论点 1"),
    (420, 720, "核心论点 2"),
    # ...
]

for start, end, label in ranges:
    chunk = " ".join([s['text'] for s in subs if start <= s['s'] <= end])
    print(f"\n【{label}】")
    print(chunk[:800])  # 截断防溢出，足矣理解完整论证
```

## 效果

- 采样大纲 → 获取结构骨架和时间边界 (~400 tokens)
- 时间窗口聚合 → 每个窗口完整连续文本 (~600-1000 tokens/窗口)
- 总计 ~4000-8000 tokens 即可完整理解 40 分钟对话的全部论点

## 适用场景

| 视频类型 | 段长度 | 采样够用？ | 建议 |
|---------|--------|-----------|------|
| 单人演讲 | 3-5s | ✅ 够用 | 采样大纲即可 |
| 密集对话/辩论 | 1-1.5s | ❌ 稀疏 | 采样 + 时间窗口聚合 |
| 教学/教程 | 2-4s | ✅ 通常够用 | 采样大纲 + 选择性聚合 |
